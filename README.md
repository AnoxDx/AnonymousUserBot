## VPS/Locally deploy!
```console
$ apt-get -y update
$ apt-get -y install git gcc python3-pip -y
$ git clone https://github.com/AnoxDx/AnonymousUserBot
$ cd AnonymousUserBot
$ pip3 install -U -r requirements.txt
$ cp sample.env .env
$ nano .env
```
